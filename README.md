 # 🤖 AI Resume Analyzer

An AI-powered Resume Analyzer that compares a candidate's resume with a job description and provides an ATS Score, skill matching, AI-generated feedback, and personalized resume improvement suggestions using **Google Gemini AI**.

## 📊 How It Works

1. Upload a PDF Resume.
2. Paste the Job Description.
3. Resume text is extracted from the PDF.
4. Skills are extracted from the resume.
5. ATS Keyword Matching compares the resume with the Job Description.
6. Google Gemini generates AI-powered feedback.
7. The application displays:

- 📊 ATS Score
- 💼 Resume Skills
- ✅ Matched Skills
- ❌ Missing Skills
- 💡 Improvement Suggestions
- 🤖 AI Resume Review
